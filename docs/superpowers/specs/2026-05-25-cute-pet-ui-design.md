# 萌宠风格 UI 设计方案

## 目标
美化右键菜单和聊天界面，采用自然清新色调的萌宠风格。

## 配色（自然清新）
| 用途 | 色值 |
|------|------|
| 窗口背景 | `#F5FAF5` |
| 装饰条底色 | `#E8F5E8` |
| 用户消息气泡 | `#D4E9D6` |
| AI 消息气泡 | `#FFFFFF` + `#E0EBE0` 边框 |
| 按钮主色 | `#8BC48B` |
| 按钮悬停 | `#6EAF6E` |
| 输入框边框 | `#D0E0D0` |
| 文字色 | `#3E5A3E` |
| 菜单背景 | `#FAFCFA` |
| 菜单选中 | `#D4E9D6` |

## 聊天窗口
- 顶部 40px 装饰条：🐾「罗小黑聊天」，浅绿底色 + 底部分割线
- 用户气泡右对齐：柔雾绿，圆角 16px
- AI 气泡左对齐：白色 + 浅绿边框，圆角 16px
- 输入框：圆角 20px，浅绿边框，左右 📎 和「发送」按钮
- 发送按钮：草木绿背景，白色文字，圆角 16px

## 右键菜单
- 暖白背景，圆角 8px
- 悬停项浅绿高亮
- 自定义图标

## 实现文件
- 新增 `styles.py`：集中管理 QSS
- 修改 `chat_window.py`：应用样式，添加装饰条
- 修改 `main.py`：菜单 QSS 样式
